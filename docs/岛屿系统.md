## 三、苍莽岛（幽暗密林）

### 3.1 基础信息

| 项目 | 内容 |
|------|------|
| **岛屿名** | 苍莽 |
| **游戏场景** | 幽暗密林 |
| **地貌** | 原始森林，阴暗潮湿 |
| **环境色** | 深绿 `#0f1519` |
| **解锁时间** | 0s（初始） |
| **难度** | ★★☆ |

### 3.2 背景故事

苍莽岛覆盖着亘古森林，树冠遮天蔽日，地面终年不见阳光。
林中潮湿阴暗，滋生各种蝙蝠和岩魔。

这里是修仙者初次历练的场所，危险程度较低。

### 3.3 原生怪物

| 怪物 | 代码ID | 描述 |
|------|--------|------|
| 幽蝠 | `bat` | 林中蝙蝠，成群出没 |
| 岩魔 | `rock` | 岩石化的精怪 |

### 3.4 产出资源

| 资源 | 描述 | 用途 |
