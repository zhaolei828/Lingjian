## 二、星落

### 2.1 世界结构

**星落**是一颗由**无数悬浮岛屿**组成的奇异星球。

岛屿大小不一，漂浮于虚空之中，互相引力吸引、保持距离。
每个岛屿都可以有自己的地貌、气候、生态。

```
                        ☀️ 太曜照耀
                            │
    ┌───────────────────────┼───────────────────────┐
    │                       ↓                       │
    │       ·  🏝️    ·    🏝️  ·    🏝️    ·        │
    │    🏝️    ·  🏝️  ·    ·  🏝️  ·    🏝️       │
    │       ·     🏝️     ·        ·               │
    │    🏝️  ·  【灵墟】  ·  🏝️  ·    🏝️        │
    │          · (人类家园) ·                       │
    │       🏝️    ·  🏝️  ·    ·    🏝️            │
    │    ·  🏝️    ·    🏝️  ·  🏝️    ·           │
    │                                               │
    │           【星落 · 万岛世界】                  │
    └───────────────────────────────────────────────┘
```

### 2.2 岛屿特点

每个岛屿：
- **大小不一**：小如孤峰，大如陆地
- **各有地貌**：森林、冰原、火山、荒漠皆可有
- **自带引力**：地表生物被束缚
- **独立生态**：孕育不同的生灵

### 2.3 虚空

岛屿之间是**虚空**，普通生物无法跨越。

**往来方式**：
- **传送阵**：各势力建造，最常用
- **飞舟**：跨越虚空的飞行法器
- **大能飞渡**：修为通天者可肉身飞越

## 三、主要岛屿

### 3.1 灵墟（人类家园）

**灵墟**是星落中人类聚居繁衍的核心岛屿。

| 项目 | 内容 |
|------|------|
| **性质** | 人类文明中心 |
| **特点** | 灵气适中，四季分明，适宜生存 |
| **地位** | 五派总部所在，新手出发点 |

灵墟上有：
- **天枢城**：最大的城市，五派会盟地，商贸中心
- **五派驻地**：天剑宗、玄元道、荒古门、幽冥涧、天机阁

**玩家从灵墟出发，通过传送阵前往各个岛屿历练。**

### 3.2 历练诸岛（游戏场景）

灵墟之外的虚空中，散布着无数岛屿。
其中有**九座**已被探索并建立传送阵，供修仙者历练：

### 3.3 仙境（飞升之地）

在星落的高处，有一座传说中的岛屿——**仙境**。

| 项目 | 内容 |
|------|------|
| **名称** | 云霄仙境 / 九天 |
| **特点** | 灵气极度充沛，仙气缭绕 |
| **地位** | 修仙者飞升的终点 |

**修仙的终极目标**：修为大成，飞升仙境，成就仙道。

游戏中的「昆仑仙境」场景，就是仙境的外围区域。
真正的仙境核心，只有飞升者才能进入。

### 3.4 冥府（亡魂归处）

在星落的深处，有一座阴森的岛屿——**冥府**。

| 项目 | 内容 |
|------|------|
| **名称** | 幽冥岛 / 黄泉 |
| **特点** | 阴气浓重，亡魂聚集 |
| **地位** | 死后灵魂的归处 |

传说：
- 生灵死后，灵魂会穿越虚空，来到冥府
- 冥府有轮回之门，决定亡魂去向
- 幽冥涧与冥府有某种联系

## 八、扩展性

### 8.1 无限可能

星落的岛屿是**无限的**，随时可以添加：
- **新岛屿** = 新地图/新场景
- **新势力** = 其他岛屿上的组织
- **新剧情** = 探索新岛屿的故事
- **星际扩展** = 前往其他星球

### 8.2 扩展示例

> "探险者发现了一座从未被记载的岛屿，那里有上古文明的遗迹..."
> "仙境传来消息，通往更高层天的道路即将开启..."
> "冥府出现异动，亡魂开始涌向人间..."
