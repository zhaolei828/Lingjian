# 五派系统设计

> 基于星落世界观，五大修仙势力皆驻于灵墟，各有传承、各有所长。

---

## 二、天剑宗

### 2.1 基础信息

| 项目 | 内容 |
|------|------|
| **代码ID** | `sword` |
| **名称** | 天剑宗 |
| **驻地** | 灵墟 · 万剑崖 |
| **道统** | 剑修 |
| **理念** | 以身化剑，唯快不破 |
| **特点** | 高攻速、穿透、灵活 |

### 2.2 背景故事

天剑宗立于万剑崖之上，崖壁插满历代祖师遗剑，剑气冲霄。

宗门传承「御剑术」，讲究人剑合一，以气驭剑，剑出如电。
天剑宗弟子追求剑道极致——**唯快不破**。

### 2.3 战斗风格

- **攻击方式**：御剑攻击，发射飞剑
- **核心优势**：攻击频率最高（0.5s），移动速度最快（160）
- **战斗策略**：快速走位，持续输出，风筝流打法

### 2.4 技能体系

| 技能ID | 名称 | 效果 | 图标 |
|--------|------|------|------|
| `sword_mult` | 万剑归宗 | 飞剑数量 +1 | ⚔️ |
| `sword_spd` | 御剑术 | 攻速 +20% | 🌪️ |
| `sword_pierce` | 青莲剑歌 | 飞剑穿透 +1 | 🗡️ |

**技能特色**：多剑、快攻、穿透

### 2.5 代表人物

- **祖师**：剑仙李太白（传说）
- **当代掌门**：青云剑君
- **知名弟子**：白衣剑客、凌霄剑童

宗门传承上古「五行大道」，擅长操纵金木水火土五行之力。

- **攻击方式**：释放五行法术，造成范围伤害
- **核心优势**：单次伤害最高（25），有范围效果
- **战斗策略**：蓄力爆发，清怪效率高

| `mage_thunder` | 九天神雷 | 攻击20%几率触发落雷 | ⚡ |

**技能特色**：范围、爆发、元素

### 3.5 代表人物

- **祖师**：玄元真人（传说）
- **当代掌门**：赤焰道尊
